主要记录一下自己折腾家庭游戏系统的想法。

现在的想法是买 Apple TV + Steelseries Nimbus手柄 + 还需要折腾一些软件平台。

街机系统

可以考虑买一台 山寨supreme网红机子 rom也机子都有了

自己玩的话 可以参考下面的软件平台 全部可以实现

主机系统

PS2/Wii/3DS等等很多近代比较游戏的模拟器并没有集成到山寨supreme网红机子或者Retroarch中 但是可以参考我软件分享那的Wii/3DS模拟器 只能在PC/Mac的桌面环境玩

手柄

定制一个自己的街机系统 我觉得最重要的是解决手柄的问题 比如 PS4 和PS4 手柄之间的联动 不是先开电脑 然后用键鼠打开模拟器 再调好分辨率这么多步骤 应该是拿起手柄就可以玩

又或者是一台掌机 本身就有手柄 也是开机立刻玩

因此我觉得 适合做街机的系统有 山寨supreme网红机子/3DS破解 + Retroarch/PSVTV破解 + PS4手柄 + Retroarch / SHeild TV + Retroarch / Apple TV + Steelseries Nimbus手柄 + Retroarch + 开发者账号签名

以下是软件平台的整理

Retroarch（街机用 全平台支持）

这个的衍生系统Lakka，而Lakka的集成方Sx05re又在国内比较火，代表文章是S905怀旧主机的帖子。

使用这个玩街机系列的东西的确是非常好的选择，如果不想在安卓盒子/Mac等等上搞，买一台专门的机子也是很好的选择。比如 抖音的网红机子 supreme

supreme外型不错 据说原型机是外国的一个大神制作 而且同屏输出 还自带手柄 用于联机的确是非常好的选择 目前就是系统比较差 用月光宝盒（Sx05re衍生物）如果系统不是山寨的话入一部好一些收藏也是很好的选择

目前在Mac/安卓盒子上使用，高度适配手柄 目前最好的街机平台（不支持PS2/3DS/Wii之后的模拟器）。

OpenEmu（街机用）

这个聚合也是做得相当好 但是操作也没有适配手柄 属于Mac下比较好的聚合平台。比小鸡强太多

Provenance

一款支持多种游戏系统的模拟器，可以在 Apple TV 上运行。需要自己编译源代码。

但是更加看好Retroarch

小鸡（街机用 iOS/安卓支持）

属于山寨集成 但是也算做得可以 但是在盒子用的话也没有适配手柄 但可以用于下载一些rom 这个盗版估计只能在中国做了。一些存档分享功能也是做到了，就是界面有点难看

最大的缺点是只能手机用吧 用手机玩街机的游戏 不太合适

IGN史上满分游戏

IGN（Imagine Games Network）是一间多媒体和评论网站，主要对象为电子游戏。其对游戏评分是游戏质量的重要参考。
自2005年重组之后，IGN完善了评分标准，已经很少给游戏满分了。
