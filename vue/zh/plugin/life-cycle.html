# 生命周期

# ready

  • 类型: AsyncFunction
  • 作用域：dev|build
module.exports = {
  async ready() {
    // ...
  }
}

提示

ready 钩子在应用初始化之后，并在某些特定的函数式 API 执行之前执行。这些函数式 API 包括：

  • clientDynamicModules
  • enhanceAppFiles

# updated

  • 类型: Function
  • 作用域：dev
module.exports = {
  updated() {
    // ...
  }
}

# generated

  • 类型: AsyncFunction
  • 作用域：build

在生产环境的构建结束后被调用，生成的页面的路径数组将作为该函数的第一个参数。

module.exports = {
  async generated (pagePaths) {
    // ...
  }
}
